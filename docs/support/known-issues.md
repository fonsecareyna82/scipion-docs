# Known Issues and Workarounds

Use this page to document problems that are already known, especially when there is a useful workaround even before a full fix exists.

This page becomes more valuable over time as support discussions and bug reports reveal recurring patterns.

---

## Where to look for current signals

In addition to this page, check the public ScipionWeb support channels:

- [ScipionWeb issue list](https://github.com/fonsecareyna82/ScipionWeb/issues)
- [ScipionWeb Discussions](https://github.com/fonsecareyna82/ScipionWeb/discussions)

This documentation page is useful for curated summaries and workarounds, while GitHub usually reflects the most recent raw activity.

---

## What belongs here

Typical candidates:

- recurring installation problems
- environment-specific failures
- temporary workarounds for runtime issues
- common UI confusion points that are not yet fully solved
- behavior affected by version mismatches

---

## Good entry structure

Each known issue should ideally include:

- a short title
- affected version or environment
- symptoms
- root cause if known
- workaround
- issue or discussion reference when available

---

## Example structure

```text
Issue:
Affected versions / environments:
Symptoms:
Known cause:
Workaround:
Tracking link:
```

---

## Why this page matters

A well-maintained known-issues page:

- reduces duplicate reports
- helps users recover faster
- improves trust because problems are acknowledged openly
- feeds back into troubleshooting and release notes

---

## Recommendation

Whenever the same problem appears repeatedly in support discussions or issues, turn it into a short known-issue entry here.
